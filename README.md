#  Smart Voting System using Facial Recognition

A secure and efficient web-based voting platform built using **Flask (Python)** for the backend, **React.js** for the frontend, and **MongoDB** for data storage.  
It uses **Face Recognition**, **CAPTCHA Verification**, and **Password Hashing** to ensure voter authenticity and prevent duplicate voting.

---

##  Features

- ✅ Facial Recognition-based Voter Authentication  
- ✅ CAPTCHA Verification to prevent bots  
- ✅ Password Hashing using Bcrypt  
- ✅ Voter Registration, Login & Voting Modules  
- ✅ Real-time Election Control (Open/Close)  
- ✅ Automatic Vote Counting & Result Display  
- ✅ MongoDB Integration for storing Voter, Candidate & Vote data  
- ✅ Secure Cross-Origin Communication using CORS
